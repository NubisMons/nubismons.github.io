基本概念

  1. 随机试验

    1. 可以在相同的条件下重复进行.
    2. 每次实验的结果可能不止一个,且能事先明确所有结果.
    3. 进行一次试验之前不能确定哪一个结果会出现.

  2. 样本空间

    随机试验所有可能结果的集合

  3. 基本事件

    由一个样本点组成的单点集

  4. 必然事件

    每次试验中都必然发生的事件(S事件)

  5. 不可能事件

    每次试验中都不可能发生的事件,也就是一个空集

事件的关系与运算

  1. 包含

    $A \subset B$,表示A发生B必然发生

  2. 和事件

    $A \bigcup B$,表示A与B至少有一个发生

  3. 积事件

    $A \bigcap B$或$AB$,表示A与B同时发生

  4. 差事件

    $A - B$,表示A发生而B不发生

  5. 互斥事件

    $AB = \varnothing $

  6. 对立事件

    $\bar{A} = S - A$ 表示A不发生
